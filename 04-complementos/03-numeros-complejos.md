# Números Complejos

## Introducción

Los números complejos surgen de la necesidad de resolver ecuaciones que no tienen solución dentro del conjunto de los números reales.

Por ejemplo:

```text
x² + 1 = 0
```

Despejando:

```text
x² = -1
```

En los números reales no existe un número cuyo cuadrado sea negativo.

Para resolver este problema se introduce la unidad imaginaria.

---

# Unidad Imaginaria

## Definición

La unidad imaginaria se representa por:

```text
i
```

y se define como:

```text
i² = -1
```

De esta definición se derivan:

```text
√(-1) = i
```

---

## Potencias de i

```text
i¹ = i
```

```text
i² = -1
```

```text
i³ = -i
```

```text
i⁴ = 1
```

A partir de aquí el patrón vuelve a repetirse.

---

# Definición de Número Complejo

Un número complejo es una expresión de la forma:

```text
z = a + bi
```

donde:

* a = parte real.
* b = parte imaginaria.
* i = unidad imaginaria.

---

## Ejemplos

```text
3 + 2i
```

```text
5 - i
```

```text
-4 + 7i
```

```text
2
```

```text
6i
```

---

# Forma Algebraica

La forma algebraica de un número complejo es:

```text
z = a + bi
```

Ejemplo:

```text
z = 4 + 3i
```

Parte real:

```text
Re(z)=4
```

Parte imaginaria:

```text
Im(z)=3
```

---

# Operaciones con Números Complejos

## Suma

Se suman las partes reales y las partes imaginarias por separado.

### Ejemplo

```text
(3+2i)+(4+5i)
```

```text
=(3+4)+(2i+5i)
```

```text
=7+7i
```

---

## Resta

Se restan las partes reales y las partes imaginarias.

### Ejemplo

```text
(5+3i)-(2+i)
```

```text
=(5-2)+(3i-i)
```

```text
=3+2i
```

---

## Multiplicación

Se aplica la propiedad distributiva.

### Ejemplo

```text
(2+i)(3+2i)
```

```text
=6+4i+3i+2i²
```

```text
=6+7i+2(-1)
```

```text
=4+7i
```

---

## División

Para dividir números complejos se utiliza el conjugado.

### Ejemplo

```text
      3+i
-------------
      2-i
```

Multiplicamos numerador y denominador por:

```text
2+i
```

Resultado:

```text
11+5i
-------
  5
```

o equivalentemente:

```text
11/5 + i
```

---

# Conjugado

## Definición

El conjugado de:

```text
a+bi
```

es:

```text
a-bi
```

Se representa por:

```text
z̅
```

---

## Ejemplo

Si:

```text
z = 4+3i
```

entonces:

```text
z̅ = 4-3i
```

---

# Módulo

## Definición

El módulo o valor absoluto de un número complejo es la distancia desde el origen hasta el punto que representa al número complejo.

Se denota:

```text
|z|
```

---

## Fórmula

```text
|z| = √(a²+b²)
```

---

## Ejemplo

Si:

```text
z = 3+4i
```

Aplicando:

```text
|z| = √(3²+4²)
```

```text
|z| = √25
```

```text
|z| = 5
```

---

# Forma Polar

## Definición

Un número complejo también puede expresarse mediante su módulo y un ángulo.

La forma polar es:

```text
z = r(cosθ + i·senθ)
```

donde:

* r = módulo.
* θ = argumento.

---

## Ejemplo

Si:

```text
r = 5
```

```text
θ = 53.13°
```

entonces:

```text
z = 5(cos53.13° + i·sen53.13°)
```

---

# Forma Exponencial

Aplicando la fórmula de Euler:

```text
e^(iθ)=cosθ+i·senθ
```

la forma polar puede escribirse como:

```text
z = re^(iθ)
```

---

## Ejemplo

```text
z = 5e^(i53.13°)
```

---

# Aplicaciones

Los números complejos se utilizan en:

* Telecomunicaciones.
* Física.
* Procesamiento de señales.
* Matemática avanzada.
