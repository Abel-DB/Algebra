# Productos Notables

## Introducción

Los productos notables son multiplicaciones algebraicas que siguen reglas fijas y permiten obtener resultados de forma rápida sin realizar la multiplicación término por término.

Su estudio facilita la simplificación de expresiones algebraicas y constituye una base importante para la factorización.

---

# Definición

Se denominan productos notables a ciertas multiplicaciones de polinomios que presentan una estructura particular y cuyo resultado puede obtenerse mediante fórmulas conocidas.

---

# Cuadrado de un Binomio

## Caso 1: Suma

Fórmula:

```text
(a+b)² = a² + 2ab + b²
```

### Ejemplo

```text
(x+3)²
```

Aplicando la fórmula:

```text
(x+3)² = x² + 2(x)(3) + 3²
```

```text
(x+3)² = x² + 6x + 9
```

---

## Caso 2: Diferencia

Fórmula:

```text
(a-b)² = a² - 2ab + b²
```

### Ejemplo

```text
(x-4)²
```

Aplicando:

```text
(x-4)² = x² - 8x + 16
```

---

# Cubo de un Binomio

## Caso 1: Suma

Fórmula:

```text
(a+b)³ = a³ + 3a²b + 3ab² + b³
```

### Ejemplo

```text
(x+2)³
```

Resultado:

```text
x³ + 6x² + 12x + 8
```

---

## Caso 2: Diferencia

Fórmula:

```text
(a-b)³ = a³ - 3a²b + 3ab² - b³
```

### Ejemplo

```text
(x-2)³
```

Resultado:

```text
x³ - 6x² + 12x - 8
```

---

# Suma por Diferencia

## Fórmula

```text
(a+b)(a-b)=a²-b²
```

### Ejemplo

```text
(x+5)(x-5)
```

Aplicando:

```text
x²-25
```

---

# Binomios con Término Común

## Fórmula

```text
(x+a)(x+b)=x²+(a+b)x+ab
```

### Ejemplo

```text
(x+2)(x+5)
```

Aplicando:

```text
x²+(2+5)x+(2)(5)
```

```text
x²+7x+10
```

---

# Triángulo de Pascal

El Triángulo de Pascal permite obtener los coeficientes de las potencias de un binomio.

Primeras filas:

```text
          1
        1   1
      1   2   1
    1   3   3   1
  1   4   6   4   1
```

Por ejemplo:

```text
(a+b)^4
```

utiliza los coeficientes:

```text
1   4   6   4   1
```

y resulta:

```text
a⁴ + 4a³b + 6a²b² + 4ab³ + b⁴
```

* Resolución de ecuaciones.
